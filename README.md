# BESS Analysis Tool (Vercel Ready)

Om dit te publiceren via Vercel:

1. Kopieer de code uit de online editor.
2. Plak deze in `src/App.tsx` (overschrijf alles).
3. Upload deze hele map naar een nieuwe GitHub Repository.
4. Ga naar Vercel.com -> New Project -> Importeer je GitHub repo.
5. Klik 'Deploy'. Vercel doet de rest (installeren en bouwen).
